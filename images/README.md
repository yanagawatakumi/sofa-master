# 画像ファイルの配置ガイド

## rooms/ フォルダ

プリセットの部屋画像（3枚）が格納されています。

### uploads/ サブフォルダ
- ユーザーがアップロードした部屋画像の一時保存先
- 合成で使用後、サーバーが自動削除する
- 手動でファイルを配置する必要はない

### 推奨仕様
- **形式**: JPEG, PNG, WebP
- **内容**: ソファを配置したい空間が写っているリビングルームなどの画像

## sofas/ フォルダ

プリセットのソファ画像（3枚）が格納されています。

### 推奨仕様
- **形式**: PNG（背景透過推奨）、JPEG, WebP
- **内容**: 正面または斜めから撮影されたソファ画像
